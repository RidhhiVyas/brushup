# Brushup

Brush up a skill in a few minutes a day. A lightweight microlearning app with bite-sized lessons and quick quizzes across topics like money, focus, mindfulness, and communication.

## Features
- Bite-sized lessons with quick-check quizzes
- Daily streak tracking
- Progress saved on your device
- Works offline once loaded
- Installable to your phone's home screen (PWA)

## Live app
https://YOUR-USERNAME.github.io/brushup/

## How to use
Open the link above on your phone. In Chrome (Android) tap the menu and choose "Install app"; in Safari (iPhone) tap Share then "Add to Home Screen."

## Editing lessons
All content lives in the `TOPICS` list near the top of the script in `index.html`. Each topic has cards that are either a reading card or a quiz card — copy the existing pattern to add your own.

## Tech
A single self-contained `index.html`: HTML, CSS, and JavaScript in one file, plus a web manifest and service worker for offline use and installability. No build step, no dependencies.
